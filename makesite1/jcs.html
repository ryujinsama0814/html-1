자바스크립트의 개념

Ecma International의 프로토타입 기반의 프로그래밍 언어로, 스크립트 언어에 해당된다. 특수한 목적이 아닌 이상 모든 웹 브라우저에 인터프리터가 내장되어 있다. 오늘날 HTML, CSS와 함께 웹을 구성하는 요소 중 하나다. HTML이 웹 페이지의 기본 구조를 담당하고, CSS가 디자인을 담당한다면 JavaScript는 클라이언트 단에서 웹 페이지가 동작하는 것을 담당한다. 웹 페이지를 자동차에 비유하자면, HTML은 자동차의 뼈대, CSS는 자동차의 외관, JavaScript는 자동차의 동력이라고 볼 수 있다.

Ecma International's prototype-based programming language, which is a scripting language. All web browsers have a built-in interpreter, except for special purposes. Today, along with HTML and CSS, it is one of the elements that make up the web. While HTML is responsible for the basic structure of a web page and CSS is responsible for design, JavaScript is responsible for operating the web page on the client side. If you compare a web page to a car, HTML can be seen as the skeleton of a car, CSS as the appearance of a car, and JavaScript as the engine of the car.
