css의 개념

HTML 등의 마크업 언어로 작성된 문서가 실제로 웹사이트에 표현되는 방법을 정해주는 스타일 시트 언어. CSS의 C가 Cascading[1]의 약자인데, 이는 상위 요소의 스타일 속성을 자손 요소들에게 상속시켜주는 모습이 DOM 트리구조에서 마치 폭포수처럼 내려가는 모습을 닮았기 때문이다. 다만 예외로 마진, 패딩, 보더 등의 박스모델 관련 속성은 상속되지 않는다. 물론 inherit 값을 줘서 강제로 상속시킬 순 있다.

A style sheet language that dictates how documents written in a markup language such as HTML are actually presented on a website. CSS C is an abbreviation for Cascading[1] because the inheritance of the style properties of the parent element to the child elements resembles a waterfall in the DOM tree structure. However, as an exception, box model related properties such as margin, padding, and border are not inherited. Of course, you can force inheritance by giving the inherit value.
